### Core Technologies
- **Keras 3**: Multi-backend deep learning framework
- **Keras Hub**: Pre-trained model library
- **JAX**: High-performance numerical computing (recommended)
- **Gemma Models**: Google's open-source LLMs (2B and 7B variants)

- **Google Generative AI**: Additional model access via API
- **NumPy/Pandas**: Data manipulation
- **Jupyter**: Interactive development

- **Model**: Gemma 1.1 Instruct 2B EN
- **Parameters**: 2 billion
- **Type**: Instruction-tuned, decoder-only transformer
- **Size**: ~5GB download
- **Context**: 4096 token sliding window
- **Vocabulary**: 256,000 tokens
